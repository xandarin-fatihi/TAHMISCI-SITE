# Local Panel Kopyasi

Bu klasor sadece local icerik kontrolu icindir. Canli yayina yuklenmemelidir.

- Giris local PIN ile yapilir: `2135`.
- Backend onayi gerekmez; panel localStorage uzerinden kontrol ve deneme icin acilir.
- Isterseniz local testte paneli `local-inceleme için/panel-local/panel.html?backend=http://localhost:8080` seklinde acip backend verisini okumayi deneyebilirsiniz.
- Degisiklikleri tarayicinin `localStorage` alanina kaydeder.
- Canli yayina alirken bu klasoru yuklemeyin.

Canli/admin yayinda kullanilacak guvenli panel klasoru: `panel/`
