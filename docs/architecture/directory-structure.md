# Klasör Yapısı

- `apps/website`: public Tahmisçi sitesi
- `apps/admin`: admin paneli
- `apps/recipe`: `/recete/` reçete ve barista ekranı
- `apps/qr-menu`: `/qr-menu/` QR menü
- `apps/auth`: login ve şifre sıfırlama sayfaları
- `apps/api`: Express API
- `public/assets`: ortak production görsel, video, font ve marka varlıkları
- `data/seeds`: takip edilen başlangıç menü, reçete ve site verileri
- `data/templates`: Excel import şablonları
- `storage`: takip edilmeyen lokal/runtime store, medya ve backup alanları
- `docs`: mimari, geliştirme, deployment, API, özellik ve tarihçe belgeleri
